[PRACTICA 2; REDES DE COMPUTADORAS]

El ejercicio 1 y 2 de la practica 2 de redes de computadoras se deben compilar juntas, puesto que el ejercicio 2
lo que realiza es encriptar un archivo dado con el metodo Cesar dado un 'offset' dado, ejemplo:

$ ./<nombre-ejecutable> <archivo> <offset>

Para compilar basta hacer lo siguiente:

$ gcc Ejercicio_1_LARE Ejercicio_2_LARE -o <nombre-ejecutable>

Cabe recalcar que puede soltar una Advertencia, pero a pesar de ello el ejecutable se crea sin ningun tipo de problema.
Existe un archivo .h para el ejercicio 1, esto con el fin de poder importar dicho codigo de forma sencilla, por lo que
no debe ser eliminado.

El ejercicio 3, se encuentra a partir de la linea de codigo 114 del archivo server.c
